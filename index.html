🌱 Min Frö- och Såddloggbok

Med Firebase - Synkas automatiskt mellan alla enheter!

🔧 Firebase Setup - Följ dessa steg:

  1. Gå till Firebase Console
  2. Klicka på "Add project" (Lägg till projekt)
  3. Namnge projektet: fro-loggbok
  4. Inaktivera Google Analytics (behövs inte)
  5. Klicka "Create project"
  6. När projektet är skapat, klicka på "Web" ikonen (</>)
  7. Ge appen ett namn: Frö-loggbok
  8. Kopiera konfigurationsobjektet (firebaseConfig)
  9. Klistra in värdena nedan:

🔒 Aktivera Firestore Database:

  1. I Firebase Console, gå till "Build" → "Firestore Database"
  2. Klicka "Create database"
  3. Välj läge: "Start in test mode" (för enkelhetens skull)
  4. Välj location: "europe-west1" (närmast Sverige)
  5. Klicka "Enable"

👤 Aktivera Authentication (för inloggning):

  1. I Firebase Console, gå till "Build" → "Authentication"
  2. Klicka "Get started"
  3. Under "Sign-in method", klicka på "Email/Password"
  4. Aktivera "Email/Password" och klicka "Save"
  5. Gå till fliken "Users"
  6. Klicka "Add user" och skapa ditt konto
  7. I "Firestore Database" → "Rules", ersätt med:
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /{document=**} {
          allow read, write: if request.auth != null;
        }
      }
    }
  8. Klicka "Publish"
  9. Nu krävs inloggning för att använda appen!
